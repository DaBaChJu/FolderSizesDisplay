
Programme pour afficher la liste des fichiers et dossiers contenus dans un dossier source,
en affichant leur tailles, et permettant de trier la liste pour chaque dossier par nom ou
par taille, ascendant ou descendant (case à cocher 3 choix).
Dossier source obtenu par cliquer-déplacer dans l'interface, ou par argument de ligne de 
commande (ou clique-déplacer d'un fichier sur l'exécutable).
Expérimentation de code commenté et nommé en français, fonction de base ok, fonctionnalités
expérimentales non terminées.

sizes, and allowing to sort the list within each folder by filename or filesize, ascending
or descending (3-choices checkbox).
Source folder set by drag&drop in the interface, or by command-line argument (or drag&drop
of a file on the exe).
